IRON AGE - By James Benjamin Brimelow Gorman (Jamie Gorman)

Enter File > Executable and run the exe to run the program.

To use the program, select generate and choose a location for a file
to be generated at, then select "RANDOM" or "EXISTING" and "GENERATE"

After this has finished, use the start page and load in the file on
the "SIMULATE" setting to begin the simulation. areas can be clicked
to see values relevant to them, and the ">" keys are used to change
the speed at which the program runs.

The EPQ documentation can be found at File > EPQ development diary
